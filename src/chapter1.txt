 1.1.INTRODUCTION
This chapter gives the foundation of discrete mathematics. Much of discrete 
mathematics is devoted to the study of discrete structures, which are used to 
represent discrete objects. All discrete structures are built up from sets, which 
are collection of objects, used extensively in counting ; relations, which are 
set of ordered pairs that represent relationships between objects ; graphs, 
which are set of vertices & connected edges.
   In this chapter we cover some of the basic set languages and notations 
that will be used throughout the text. Venn diagrams will be introduced in 
order to give clear picture of set theory.

  1.4. MULTI-SETS AND MULTIPLICITY
If a set contains exactly m distinct elements where m denotes some non- 
negative integers, then the set is said to be a finite set otherwise the set is 
said to be an infinite set.
   For example, the set {1, 2, 5, 7, 9} is a finite set whereas N, the set of 
natural number is infinite set.
   As we have discussed above that we can define a set by different ways. 
Now, we define a multiset which is used to define a collection of objects which 
are not necessarily distinct.
   For example {1, 1, 2, 2, 2}, {a, b, c}, { }, {1, 1} are all multisets.
   Multiplicity of a Set : Number of times an element appearing in a set 
is called multiplicity of that element. As in the above first example {1, 1, 2, 
2, 2} multiplicity of 1 is 2 & multiplicity of 2 is 3.
   Definition : The sum of the multiplicities of the elements in the sets 
P and Q is denoted by multiplicity in the set P + Q.
